    Class OrderService

    Defined in: src/coffee/services/orders.coffee
    Inherits: BaseService

    Overview

    Define a OrderService to interact with the HTTP orders endpoint.

    An order is the final state of a cart, usually created after a checkout process has been completed.

    Examples:

    service = client.orders()
service.byId('123').fetch()
.then (result) ->
  service.byId('123').update
    version: result.body.version
    actions: [
      {
        action: 'changeOrderState'
        orderState: 'Complete'
      }
    ]
