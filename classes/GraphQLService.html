    Promise ::query(body) Source

    Save a new resource defined by the Service by passing the payload Object.

    Examples:

    client.graphql.query
  query: """
    query Sphere {
      channels(sort: "createdAt asc", limit: 2) {
        total,
        results {
          createdAt, key
        }
      },
      products(limit: 1, skus: ["sku1444727363571_7"]) {
        ...StagedProduct,
        ...CurrentProduct
      }
    }
    fragment Product on ProductData {
  skus, name(locale: "en")
}
    fragment StagedProduct on ProductQueryResult {
  results {
    id, masterData { staged { ...Product } }
  }
}
      fragment CurrentProduct on ProductQueryResult {
    results {
      id, masterData { current { ...Product } }
    }
  }
"""

    Parameters:

    • (Object) body — Object The payload described by the related API resource as JSON

    Throws:

    • (Error) — Throws an Error if body is missing

    Returns:

    • (Promise) — a Promise, fulfilled with an Object or rejected with an instance of an HttpError or SphereError
