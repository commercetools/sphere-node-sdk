    Class StateService

    Defined in: src/coffee/services/states.coffee
    Inherits: BaseService

    Overview

    Define a StateService to interact with the HTTP states endpoint.

    A State represents a state of a particular entity (defines a finite state machine). States can be combined together by defining transitions between each state, thus allowing to create work-flows.

    Examples:

    service = client.states()
service.byId('123').fetch()
.then (result) ->
  service.byId('123').update
    version: result.body.version
    actions: [
      {
        action: 'setName'
        name:
          en: 'Foo'
      }
    ]
